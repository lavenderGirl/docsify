# 2018-不畏将来，勇于改变。
与其呆在原处一成不变，未知事物更俱吸引，因未知并不惧，突破常规，惊喜都在来的路上，只要一路向前，迟早碰上。

当坐在上海1933老场坊的空中花园晒着太阳闲聊时，那时我说“我们要成为更好的自己，对生活多点心思，对工作多些热情，对身边的人更多体谅与爱。”自15年毕业来同程实习工作三年，三年的时间足够认识这座城市，干净舒适而美好，那些美好的事物依然犹存心间。骑行独墅湖看日出；天平山赏红枫；诚品书店品书；灵白线徒步...无论是一个人还是一些人，其实都是想去遇见更好的自己。

3月16日，当我辞职同程，决意来上海，其实内心也是几番纠结，恐惧与害怕让我想要退缩。感谢自己不畏将来的勇气，勇敢走出舒适区来到上海，现在回头想想，当初的纠结与害怕无非就是害怕工资没有当下的高；环境没有当下的好；生活没有当下舒适。却忘了为何想要辞职，难道不是因为止步不前的能力和工作不开心的氛围？与其待在原处抱怨，难道不是越早改变收益越大么。个人觉得没有什么比让自己开心快乐更重要，至于工资的高低取决于自己的能力，如果工资没达到预期，说明自己的能力还没达到，先好好修炼自己。你想要的东西都会因为你的改变与努力送到你的面前。

2018只要心向阳光，一切都可重新开始。感谢自己的一路往前，在不知觉之时正在慢慢点亮我的人生。
